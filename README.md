# Viper Filter

Viper Filter is a professional file filtering and text analysis application designed for efficiently searching, filtering, and extracting specific content from large text files.

## Features

- **Multi-keyword Filtering**: Filter lines containing any or all specified keywords, separated by commas, semicolons, or newlines
- **Regular Expression Support**: Full regex pattern matching with error handling
- **Duplicate Removal**: Automatically removes duplicate lines from filtered results and sorts them alphabetically
- **Large File Support**: Progress indication with cancel capability for processing large files
- **Keyword Highlighting**: Visual highlighting of matched keywords within filtered results
- **Multiple Output Options**: Copy selected text, copy all results, or export filtered results to a new file
- **Persistent Settings**: Remembers recent files, window geometry, and application preferences
- **User-Friendly Interface**: Professional dark "matrix" themed interface with collapsible sidebar

## System Requirements

- Windows operating system
- No additional software dependencies required

## Usage

1. **Load a File**: Click "Browse" or drag and drop a text file into the application
2. **Specify Filter Criteria**: Enter keywords in the search field. Keywords can be separated by commas, semicolons, or newlines
3. **Configure Filter Options**:
   | Option | Description
   | ---- | ------ |
   | ANY (default) | Show lines containing any of the specified keywords
   | ALL | Show only lines containing all specified keywords
   | Regex mode | Treat keywords as regular expression patterns
4. **Filter**: Click the "Filter" button to process the file and display matching lines
5. **Results**: Filtered results appear in the "Filtered" tab with matched keywords highlighted

## Available Actions

- **Copy Selected**: Copy only the currently selected text from results
- **Copy Results**: Copy all filtered results to the clipboard
- **Export**: Save filtered results to a new text file
- **Recent Files**: Access previously opened files from the sidebar

## Download
